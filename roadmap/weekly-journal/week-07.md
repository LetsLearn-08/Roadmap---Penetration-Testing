## üìñ Overview
Cryptography is the science of securing information by transforming it into unreadable formats for unauthorized users. It ensures:
- [Confidentiality](https://en.wikipedia.org/wiki/Confidentiality)
- [Integrity](https://en.wikipedia.org/wiki/Data_integrity)
- [Authentication](https://en.wikipedia.org/wiki/Authentication)
- [Non‚Äërepudiation](https://en.wikipedia.org/wiki/Non-repudiation)

---

## üß© Core Concepts
- **[Plaintext](https://en.wikipedia.org/wiki/Plaintext)** ‚Üí Original readable message  
- **[Ciphertext](https://en.wikipedia.org/wiki/Ciphertext)** ‚Üí Scrambled output after encryption  
- **[Encryption](https://en.wikipedia.org/wiki/Encryption)** ‚Üí Converting plaintext into ciphertext using an algorithm + key  
- **[Decryption](https://en.wikipedia.org/wiki/Decryption)** ‚Üí Reversing ciphertext back into plaintext  
- **[Key](https://en.wikipedia.org/wiki/Cryptographic_key)** ‚Üí Secret value used in encryption/decryption

## üîê Types of Cryptography
- **[Symmetric Key Cryptography](https://en.wikipedia.org/wiki/Symmetric-key_algorithm)** (e.g., [AES](https://en.wikipedia.org/wiki/Advanced_Encryption_Standard), [DES](https://en.wikipedia.org/wiki/Data_Encryption_Standard))  
- **[Asymmetric Key Cryptography](https://en.wikipedia.org/wiki/Public-key_cryptography)** (e.g., [RSA](https://en.wikipedia.org/wiki/RSA_(cryptosystem)), [ECC](https://en.wikipedia.org/wiki/Elliptic-curve_cryptography))  
- **[Hash Functions](https://en.wikipedia.org/wiki/Cryptographic_hash_function)** (e.g., [SHA‚Äë256](https://en.wikipedia.org/wiki/SHA-2), [MD5](https://en.wikipedia.org/wiki/MD5))
